Ping Pong: O primeiro video-game criado

Esta aplicação web apresenta nosso projeto de Ping Pong para Arduino. O objetivo foi criar uma experiência de jogo física e interativa, controlada por dois jogadores. Esta página serve como um relatório dinâmico, demonstrando a construção, os componentes e o funcionamento do jogo.

Detalhes Técnicos

O projeto combina uma lógica de software robusta com funcionalidades que criam uma experiência de jogo completa e agradável. Aqui estão os principais destaques técnicos e as bibliotecas que tornaram tudo possível.

Funcionalidades Principais

  • ✓Controle Intuitivo: Raquetes controladas suavemente por potenciômetros.
  • ✓Feedback Sonoro: Sons para rebotes, pontos e eventos de jogo aumentam a imersão.
  • ✓Gerenciamento de Jogo: Funções de pausa e reinício para total controle da partida.
  • ✓Tela de Créditos: Uma tela inicial apresenta a equipe de desenvolvimento.
  • ✓Lógica de Colisão Precisa: O código detecta com exatidão as colisões da bola.

Bibliotecas Utilizadas

Para a renderização gráfica no display OLED, utilizamos duas bibliotecas essenciais da Adafruit:

Adafruit GFX Library

Biblioteca gráfica principal com funções de desenho.

Adafruit SSD1306

Driver específico para o nosso modelo de display OLED.

Como Jogar

Jogar é simples e intuitivo. Siga os passos abaixo para começar a partida. O objetivo é ser o primeiro a marcar 5 pontos!

1️⃣

Movimente a Raquete

Gire o potenciômetro ou joystick para mover sua raquete verticalmente.

2️⃣

Pause e Reinicie

Use os botões dedicados para pausar/retomar ou reiniciar o jogo a qualquer momento.

3️⃣

Marque Pontos

Rebata a bola e tente passar pela raquete do seu oponente para marcar.
